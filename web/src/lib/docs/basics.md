# Basics

Note: Semicolons are not required!

## Variables

```rust
let a = 1;
let b = "a";
let c = false;
```

## Functions

```rust
fn yap(msg) {
    print(msg);
}

yap("Hello, World!");

// Outputs
//
// Hello, World!
```

## Conditions

```rust
if 1 == 1 {
    print("yes");
}

if 1 != 2 {
    print("duh");
}

// Outputs
//
// yes
// duh
```
